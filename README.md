# 🧠 AI Task Prioritizer – Smart To-Do App

An intelligent full-stack task management system built with **Angular + Spring Boot + Spring AI (OpenAI)** that helps users decide what to do next using AI-powered prioritization.

---

## ✨ Features

- 📋 Create, update, delete tasks
- 🧠 AI-powered task prioritization (Spring AI + OpenAI)
- 🔐 Secure authentication (JWT + Cookies)
- 📊 Smart productivity suggestions
- ⚡ RESTful API backend
- 🎨 Modern responsive UI (Angular)

---

## 🤖 AI Feature (Core Highlight)

The system uses **Spring AI + OpenAI** to analyze the user’s task list and return the most important task to do next.

### Example:

> Input:["Study Angular", "Fix bug in backend", "Watch tutorial"]
> Output: Next Task: Fix bug in backend
          Reason: It blocks progress and has higher priority impact.
